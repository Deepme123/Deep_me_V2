Emotion Session Demo

대화부터 분석 카드까지 한 화면에서 확인

이 페이지는 감정 대화 WebSocket 흐름을 팀이 직접 시연하고 검증하기 위한 내부 데모 화면입니다. 대화 이벤트, 종료 시점, 분석 카드 생성, 저장된 카드 조회를 한 번에 확인할 수 있습니다.

연결 설정

인증 토큰이 있으면 입력하고, 데모 모드면 비워둔 채 연결해도 됩니다.

연결 안 됨
session_id: 없음

권장 순서: 연결 후 메시지를 2~4번 보내고, `close_ok`와 `analysis_card_ready`가 같은 흐름 안에서 오는지 확인하세요.

메시지 보내기

실제 사용자 입력처럼 한 문장씩 보내면 됩니다.

대화 화면

사용자와 AI의 최종 메시지만 정리해서 보여줍니다.

아직 대화가 없습니다. 연결한 뒤 메시지를 보내세요.

실시간 이벤트 로그

WebSocket에서 실제로 받은 원본 이벤트를 보여줍니다. 발표 중에는 `open_ok`, `close_ok`, `analysis_card_ready`를 중심으로 보면 됩니다.

이벤트가 여기에 쌓입니다.

분석 카드 결과

`analysis_card_ready`가 오면 카드 내용을 보기 쉽게 정리해서 표시합니다.

아직 분석 카드가 생성되지 않았습니다.

저장된 카드 조회

세션 종료 후 실제로 카드가 저장되었는지 다시 확인하는 영역입니다.

세션이 생기면 `저장된 카드 조회` 버튼으로 확인할 수 있습니다.
