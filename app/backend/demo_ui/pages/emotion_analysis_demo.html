감정 세션 데모

대화부터 분석 카드까지 한 화면에서 확인합니다.

감정 상담 데모 흐름을 실시간으로 점검하는 화면입니다. 연결 상태, 대화, 이벤트 로그, 분석 카드, 저장 결과를 분리된 스크롤 없이 한 번에 비교하면서 확인할 수 있습니다. 기본값은 시스템 테마를 따르고, 한 번 고른 모드는 이 브라우저에 저장됩니다.

세션 제어

연결 준비와 메시지 전송을 한 흐름에서 다루도록 묶었습니다. 연결 후 바로 대화를 이어 가며 세션 상태를 확인할 수 있습니다.

연결 전

연결 설정

토큰이 있으면 입력하고, 데모 모드에서는 비워 둔 채로도 연결할 수 있습니다.

session_id: 없음

권장 순서: 연결 후 메시지를 2~4번 보내고 마무리 유도 예시로 대화를 끝내 보세요. 모델이 종료 응답 끝에 예약 토큰을 붙이면 close_ok와 analysis_card_ready가 이어집니다. 자동 종료는 모델이 STEP 12 마무리 응답 끝에 [[CONFIRM_CLOSE]]를 붙일 때만 동작합니다. 그냥 종료는 세션만 닫고, 분석 후 종료는 세션 종료 뒤 분석 카드 생성까지 함께 요청합니다.

메시지 보내기

Enter로 전송하고, Shift+Enter로 줄바꿈할 수 있습니다.

`confirm_close`는 세션 종료와 분석 카드 생성을 함께 요청합니다. `close`는 카드 생성 없이 세션만 닫습니다.

대화 내역

사용자와 AI 메시지를 시간 순서대로 정리해 보여줍니다.

아직 대화가 없습니다. 연결한 뒤 메시지를 보내 보세요.

분석 카드 결과

분석 카드가 준비되면 영문 키 대신 한국어 라벨로 정리해 보여줍니다.

아직 분석 카드가 생성되지 않았습니다.

실시간 이벤트 로그

이벤트 타입은 유지하되, 화면에는 한국어 제목과 설명 중심으로 정리해 보여줍니다.

이벤트가 여기에 쌓입니다.

저장 카드 조회

저장된 카드를 JSON 덤프 대신 요약과 메타 정보 중심으로 읽기 쉽게 보여줍니다.

세션이 끝나면 저장 카드 조회 버튼으로 결과를 확인할 수 있습니다.
